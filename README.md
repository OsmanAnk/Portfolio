# Portfolio

Portfolio is a personal website that presents my frontend development skills, selected projects, references, and contact information.

## Live Demo
[Open live demo](https://osman-annak.developerakademie.net/Portfolio/)

## Features
- Responsive layout for desktop and mobile
- Hero section with navigation
- About me section
- Skill overview
- Project showcase with live demos and GitHub links
- Teamplayer reference section
- Contact form with validation
- Privacy policy and legal notice pages
- German and English language switch

## Tech Stack
- HTML
- CSS
- JavaScript
- PHP

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/OsmanAnk/Portfolio.git
```

2. Navigate into the project folder:

```bash
cd Portfolio
```

3. Open the project:

Open the `index.html` file in your browser or use the Live Server extension in VS Code.

> Note: The contact form requires a server environment with PHP support to send emails. The frontend validation works locally.
